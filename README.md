你可以随意的使用我编写的脚本，包括修改并商用，但是无论怎么样，你都应该遵循我添加的许可证。  
我会对我的脚本尽力增强它的兼容性、易用性和性能，并且我会给出对其做兼容适配的一些建议。  
当然，你也可以自行修改，但我希望是在你有 `JavaScript` 基础的情况下。
